# Produkt-Update.TEMU
Das Skript liest eine CSV aus PlentyMarkets ein, bereitet die Produktdaten automatisch auf, rechnet Maße um, berechnet Stückzahlen, überträgt alle relevanten Werte in eine TEMU-Excel-Vorlage und ersetzt dort alte Daten. Dadurch entsteht eine vollständig aktualisierte Importdatei für TEMU – schnell, fehlerfrei und automatisiert.
